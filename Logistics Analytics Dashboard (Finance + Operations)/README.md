# Analystics Report (Power BI + SQL + Python)

## Overview
This project showcases a **12-page Power BI report** analysing shipping performance and margin tracking for a logistics business (~8M rows of data).  
The report provides insights for **Finance, Commercial, and Operations teams**, supporting supplier negotiations and identifying loss-making jobs.

---

## Key Features
- Executive dashboard with KPIs (Revenue, Cost, Margin %).  
- Drillthrough pages for country-level and supplier-level analysis.  
- Charge breakdown visuals (fuel surcharge recovery, returns, surcharges).  
- Benchmark scatter plots comparing clients and suppliers.  
- Management-pack ready visuals for CFO and senior leadership.

---

## Tech Stack
- **Power BI** (12-page interactive dashboard, DAX measures)  
- **SQL** (fact tables for shipments and charges)  
- **Python** (CSV → Parquet → DuckDB ETL pipeline)  
- **dbt** (enrichment and fiscal calendar modelling)
